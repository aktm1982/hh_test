Порядок установки:
1. Папку honey-hunters из архива скопировать в каталог публичных документов сервера (www, htdocs, domains и т.д.)
2. В скопированном каталоге honey-hunters отредактировать файл settings/env.php: указать данные для подключения к серверу MySQL (хост, имя пользователя, пароль). Имя базы данных не менять.
3. Папку comments_db из архива скопировать в каталог data в корневом каталоге установленной базы MySQL.
4. Открыть терминал, перейти в каталог приложения - honey-hunters.
5. Выполнить команды:
    npm install - для установки зависимостей в проект;
    npm run build - для сборки проекта;
6. Запустить сервер, перейти на сайт honey-hunters

* для работы с базой данных потребуется установленное расширение mysqli для php
